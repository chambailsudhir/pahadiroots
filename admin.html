🗄️ DB Setup & Migration

Run these SQL scripts in Supabase SQL Editor

ℹ️ Run all SQL scripts below in your Supabase Dashboard → SQL Editor. Each is idempotent (safe to run multiple times).

1. Fix Column Name (categories)

⚠️ Run this once. After running, the admin panel switches from c['is active'] to c.is_active automatically.
ALTER TABLE categories RENAME COLUMN "is active" TO is_active;

2. Add Performance Indexes

CREATE INDEX IF NOT EXISTS idx_order_logs_order_id ON order_logs(order_id); CREATE INDEX IF NOT EXISTS idx_order_status_history_order_id ON order_status_history(order_id); CREATE INDEX IF NOT EXISTS idx_inventory_logs_product_id ON inventory_logs(product_id); CREATE INDEX IF NOT EXISTS idx_coupon_usage_order_id ON coupon_usage(order_id); CREATE INDEX IF NOT EXISTS idx_coupon_usage_customer_id ON coupon_usage(customer_id); CREATE INDEX IF NOT EXISTS idx_orders_customer_id ON orders(customer_id); CREATE INDEX IF NOT EXISTS idx_orders_order_status ON orders(order_status); CREATE INDEX IF NOT EXISTS idx_orders_created_at ON orders(created_at DESC); CREATE INDEX IF NOT EXISTS idx_products_category_id ON products(category_id);

3. Create Revenue Summary View (Analytics optimization)

CREATE OR REPLACE VIEW revenue_summary AS SELECT SUM(total_amount) AS revenue, COUNT(*) AS total_orders, AVG(total_amount) AS avg_order_value, COUNT(*) FILTER (WHERE order_status = 'delivered') AS delivered_count FROM orders WHERE order_status != 'cancelled';

4. Soft Delete — Add is_deleted to Products

ALTER TABLE products ADD COLUMN IF NOT EXISTS is_deleted boolean DEFAULT false; CREATE INDEX IF NOT EXISTS idx_products_is_deleted ON products(is_deleted);

5. Future: Role-Based Admin Users Table

CREATE TABLE IF NOT EXISTS admin_users ( id uuid PRIMARY KEY DEFAULT gen_random_uuid(), email text UNIQUE NOT NULL, password_hash text NOT NULL, role text NOT NULL CHECK (role IN ('super_admin','manager','support')), created_at timestamptz DEFAULT now() );
